# AI-Powered-Product-Recommendation-System-for-E-Commerce-Platforms


## 📌 Project Overview

The AI-Based Product Recommendation System is a web application developed using Python, Flask, MySQL, and Machine Learning techniques. The system provides personalized product recommendations by analyzing user interactions and combining Content-Based Filtering and Collaborative Filtering algorithms. A feedback mechanism is also included to improve recommendation quality over time.

## 🧠 Recommendation Techniques

### Content-Based Filtering
Recommends products similar to those previously viewed by the user based on product attributes such as category, brand, and description.

### Collaborative Filtering
Uses Cosine Similarity to identify users with similar interests and recommends products liked by similar users.

### Hybrid Recommendation
Combines Content-Based Filtering and Collaborative Filtering with user feedback to generate more personalized recommendations.

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/AI-Based-Product-Recommendation-System.git
```

2. Navigate to the project folder

```bash
cd AI-Based-Product-Recommendation-System
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Configure the MySQL database credentials in `Database.py`.

5. Run the application

```bash
python app.py
```

6. Open your browser

```
http://127.0.0.1:5000
```
